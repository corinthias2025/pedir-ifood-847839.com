Use o código abaixo para pagar com PIX. Clique para copiar:

  1. Abra o aplicativo do seu banco ou carteira digital.
  2. Acesse a opção de pagamento via PIX.
  3. Escolha a opção "PIX Copia e Cola".
  4. Cole o código PIX copiado no campo indicado.
  5. Confira os dados e confirme o pagamento.
  6. Guarde o comprovante para eventuais confirmações.
